# Matérias do curso de Engenharia de Computação (AA) da UNICAMP

Este repositório funciona como um índice central das disciplinas cursadas durante minha graduação. 

Cada disciplina possui seu próprio repositório, contendo materiais como trabalhos, listas de exercícios, projetos, relatórios e códigos desenvolvidos ao longo do curso.

## Como clonar (todas as matérias)

```bash
git clone --recurse-submodules https://github.com/Henrique-hpds/Materias_UNICAMP.git
```
